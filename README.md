# Ride Booking System 🚖

A full-stack ride booking web application built as a 4th semester DBMS project.

## Features
- Passenger registration and login
- Book a ride (pickup, destination, date, time)
- View booking history with status
- Cancel bookings
- Driver dashboard to view and accept rides
- Complete rides and update status

## Tech Stack
- PHP
- MySQL
- HTML & CSS
- XAMPP
- SQL Server Management Studio (SSMS)

## Database Design
Designed with 10+ tables including Passenger, Driver, Ride, Payment, etc.
